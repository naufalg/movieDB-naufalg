## Getting Started

### Installation

1. Get API Key at [http://www.omdbapi.com/](http://www.omdbapi.com/)
2. Clone the repo

```sh
git clone https://github.com/naufalg/movieDB-naufalg.git
```

3. Install NPM packages

```sh
npm install
```

4. Enter your API in `.env`

```JS
REACT_APP_OMDB_API_KEY=ENTER YOUR API;
```
